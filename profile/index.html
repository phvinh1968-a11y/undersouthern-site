Catalog Scope

Warning: An "advanced user" can submit us the releases he creates, if you give him access to "Create my releases" section. Moreover, if you have access to believe FTP account to upload your audio files, the advanced user can access it too. If you want a user creates his catalogue and submit it without your help, choose this option. If you want to keep control on what a user can distribute with us, choose instead "standard user".
Catalog scope will allow you to define which areas of the interface can be viewed by 'access-only' user accounts.
